Name Example Notes
type="button" defines a button. Should add "value" for a text
type="checkbox" defines a checkbox
type="color" is used for input fields that should contain a color
type="date" is used for input fields that should contain a date. Safari has problems with it
type="email" is used for input fields that should contain an e-mail address
type="file" defines a file-select field and a "Browse" button for file uploads
type="hidden" defines a hidden input field. A hidden field often stores what database record that needs to be updated when the form is submitted.
type="number" defines a numeric input field. Can be attribute max
type="password" defines a password field
type="radio" defines a radio button
type="range" defines a control for entering a number whose exact value is not important (like a slider control). Default range is 0 to 100. However, you can set restrictions on what numbers are accepted with the min, max, and step attributes
type="reset" defines a reset button that will reset all form values to their default values
type="search" is used for search fields (a search field behaves like a regular text field, but there is a cross on the right)
type="submit" defines a button for submitting form data to a form-handler. Needs some php to send data
type="tel" is used for input fields that should contain a telephone number
type="text" defines a single-line text input field
type="time" allows the user to select a time (no time zone)
type="url" is used for input fields that should contain a URL address
type="month" allows the user to select a month and year. Not supported in Safari
type="week" allows the user to select a week and year. Not supported in Safari
type="image" Define an image as a submit button. The path to the image is specified in the src attribute
type="datetime-local" specifies a date and time input field, with no time zone
select element is used to create a drop-down list. Also has attribute Multiple if you want to choose several
Textarea tag defines a multi-line text input control. Better specify size in css
Datalist tag specifies a list of pre-defined options for an input element
